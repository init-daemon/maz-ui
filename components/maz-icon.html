MazIcon â

MazBtn is a standalone component to load your svg icons

Basically, this component will load your SVG from your project.

Place your SVG files in a public folder (default /icons, use path prop to change it)

Get Icons Pack â

This pack is the Heroicons icons set with some others from maz-ui

Props â

Prop nameDescriptionTypeValuesDefault
srcThe source path of the SVG file - e.g: /icons/home.svgstring-undefined
pathThe path of the folder where the SVG files are stored - e.g: /iconsstring-undefined
nameThe name of the SVG file - e.g: homestring-undefined
sizeThe size of the SVG file - e.g: 1emstring-undefined
titleThe title of the SVG file - e.g: Homestring-undefined
transformSourceThe function to transform the source of the SVG file - e.g: (svg) =&gt; svg(param: SVGElement) => typeof param-(svg: SVGElement) => svg
